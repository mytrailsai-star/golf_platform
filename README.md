## Project Overview

The Golf Platform is a Python-based full-stack project developed to create an interactive web application with separate frontend and backend services.  
The frontend was built with **Streamlit** to provide a simple and user-friendly interface, while the backend was developed using **Flask** to handle APIs and server-side logic.  
Both services were deployed independently, with the frontend hosted on **Streamlit Community Cloud** and the backend hosted on **Render**.  
This project helped in understanding real-time frontend-backend communication, cloud deployment, debugging, and project structuring.

## Features

- Interactive frontend built with Streamlit
- Backend API built with Flask
- Frontend and backend deployed live
- API-based communication between services
- Clean project structure for development and deployment
- Demonstrates practical Python web development skills
